# Demo Film management application

Required Tech stack

- Node js >= V10.x
- PostGreSQL >=9.x
- yarn

#Pre-requisites

- make .env file
- copy .env.example to .env
- replace required credentials

## Installation


`yarn  install`

`yarn dev` //to run the application

`yarn dbmigrate` // to migrate the database and seed the db

`yarn docs` // to get the api Doc
